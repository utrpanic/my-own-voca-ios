# My Own Voca

## Environments
- Xcode 15.1

## Requirements

### 1. 저장하기
- 읽으면서 단어들을 종이에 적어나간다.
- 그 날의 분량이 끝나면, 단어들을 앱에 입력한다.
- 입력할 때마다, 단어 뜻을 함께 보여준다.
- 어디에서 본 단어인지, 몇 번이나 저장된 것인지 함께 표시한다.

### 2. 연습하기
- 말해보카 스타일로 랜덤하게 문제를 준다.
- 문장은 ChatGPT에게 요청한다. 고등학교 수준의 예문으로 달라고 하면 좋을 것 같다.

### 3. Local Dictionary
- `UIReferenceLibraryViewController` 사용.
